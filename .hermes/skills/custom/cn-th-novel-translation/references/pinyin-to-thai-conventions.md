# พินอิน → ไทย — กฎถอดเสียง

## หลักการทั่วไป

ใช้ระบบถอดเสียงพินอิน (Hanyu Pinyin) → ตัวอักษรไทย
อิงจากหลักการของราชบัณฑิตยสถาน + ธรรมเนียมแปลนิยายจีนในไทย

| พินอิน | เสียง | ไทย | ตัวอย่าง |
|--------|------|-----|---------|

| **d** | ไม่มีลม | **ต** | 大 dà → ต้า |

| **n** | นาสิก, ปุ่มเหงือก | **น** | 你 nǐ → หนี่ |
| **l** | ข้างลิ้น | **ล** | 来 lái → หลาย |
| **g** | ไม่มีลม, เพดานอ่อน | **ก** | 哥 gē → เกอ |
| **k** | มีลม, เพดานอ่อน | **ค** | 看 kàn → ค่าน |

### ⚠️ กับดักพบบ่อย (Common Pitfalls)

| พินอิน | ❌ ห้ามใช้ | ✅ ใช้ | เพราะ |
|--------|-----------|-------|-------|
| **shi** (是, 十, 石) | ชื่อ, ชี, ซี | **ซือ** | sh = retroflex s → ใกล้ ซ ที่สุด |
| **si** (四, 死) | ซี | **สื่อ** | s = flat → ส |
| **qi** (七, 气) | ชี | **ชี่** | q = aspirated ch → ช, i หลัง q อ่านว่า อี |
| **xi** (西, 洗) | ซี่ | **ซี** | x = alveolo-palatal → ซี |
| **zhi** (之, 只) | จี | **จือ** | zh = retroflex j → จือ |
| **chi** (吃, 迟) | ชี | **ฉือ** | ch = retroflex ch → ฉือ |
| **ri** (日) | รี | **รื่อ** | ri = retroflex r → รื่อ |
| **zi** (子) | จี | **จื่อ** | z = flat j → จื่อ |
| **ci** (此) | ชี | **ฉื่อ** | c = aspirated flat → ฉื่อ |
| **ju** (举) | จู | **จฺวี่** | u หลัง j/q/x อ่านว่า \ü\ |
| **qu** (去) | ชู | **ชวี่** | \\
| **xu** (需) | ซู | **ซวี/สวี** | \\
| **yue** (月) | ยู | **เยว่** | üe → เยว่ |
| **yuan** (远) | ยูอัน | **เยวี่ยน** | üan → เยวี่ยน |
| **yun** (云) | ยุน | **ยฺวิน** | ün → ยฺวิน |

## กฎจำง่ายสำหรับชื่อนิยาย

```
Shi → ซือ (เสมอ!)   - 石牧 = ซือมู่, 石亭 = ซือถิง
Qi  → ชี่            - 氣 = ชี่, 齊 = ฉี
Xi  → ซี            - 西 = ซี
Yu  → หยู/ยฺหวี่     - 玉 = หยู๋
```

## แนวทางปฏิบัติ

1. ถ้าสงสัยชื่อ → **หา Chinese original** ตรวจสอบเสมอ
   - แหล่ง: czbooks.net, biquge, etc.
   - ดูตัวจีนจริง → check พินอิน → ถอดไทย
2. ใน English version ชื่อจีนอาจ Anglicized (Victor Zhang, Dragon Emperor)
   - Anglicized name → แปลความหมาย
   - Chinese surname in English → ถอดพินอิน→ไทย
3. Consistency สำคัญกว่า perfection
   - ถ้าเคยแปล "Zhang" เป็น "จาง" ไปแล้ว → ใช้ "จาง" ตลอด
   - ใช้ `.hermes.md` + memory เก็บ reference
