# Quality Improvement — Novel Translation Polish Rubric

## 2-Step Polish Checklist

ใช้หลังแปล draft เสร็จ — สั่ง Hermes polish อีกรอบตาม checklist นี้

### ✅ Structure & Flow
- [ ] ตัด relative clause แบบ ENG → ใช้โครงสร้างไทย (อนุประโยค + "ที่/ซึ่ง/อัน" หรือแยกประโยค)
- [ ] เปลี่ยน passive voice → active voice ("ถูกทำ" → "เกิด...", "โดน...")
- [ ] ใช้คำเชื่อมไทย: **เมื่อ...ก็, ครั้น..., ทว่า..., เวลานี้, ฝ่าย..., แล้ว...จึง** 
- [ ] ตัด "ของ" ที่ไม่จำเป็น: "สีหน้าของซือมู่" → "สีหน้าซือมู่"
- [ ] หลากคำกริยาพูด: เอ่ย, ว่า, ตอบกลับ, ย้อน, กล่าว, อุทาน — ไม่ซ้ำ "พูด"

### ✅ Character References
- [ ] สลับการอ้างถึง: ไม่ใช่ "ชายชราชุดน้ำเงิน" ทุกย่อหน้า → ใช้ "เขาผู้นั้น", "อีกฝ่าย", "สจ๊วตเฉิง", "คนแก่ชุดน้ำเงิน"
- [ ] ไม่ทับศัพท์ชื่อ ENG → ใช้ชื่อไทย (ซือมู่ ไม่ใช่ Shi Mu)
- [ ] สมญานาม แปลความหมาย: "the blue-shirted old man" → "ชายชราสีน้ำเงิน"

### ✅ Dialogues
- [ ] ลง particles ตามชนชั้นตัวละคร:
  - ผู้ใหญ่/เจ้านาย: **"เจ้า...", "กระนั้นหรือ?", "หรือว่า"**
  - ผู้รับใช้: **"ขอรับ", "เจ้าค่ะ", "ข้าน้อย", "กระหม่อม"**
  - ชาวบ้าน/เด็ก: **"นะ", "สิ", "หรอ", "จ้า"**
- [ ] ไม่ใช้ "พูด" หรือ "กล่าว" ซ้ำกันทุกประโยค
- [ ] น้ำเสียงสม่ำเสมอตามตัวละคร

### ✅ Onomatopoeia
| ENG | ไทยที่ถูก |
|-----|----------|
| whoosh | วาบ / ฟู่ / ปรู๊ด |
| boom | เปรี้ยง / โครม |
| crack-crack | กร็อบแกร็บ |
| plop | ตุ๊บ / ตูม |
| splash | ซ่า / กระเซ็น |
| swish / slice | ฟิ้ว / วาบ |

### ✅ Terminology Consistency
- [ ] ชื่อตัวละครตรงกับ glossary.json
- [ ] ศัพท์ยุทธ์ตรงกับที่ set ไว้ (บำเพ็ญกาย, พลังแท้จริง, เส้นลมปราณ...)
- [ ] หน่วยวัดไทย: จั้ง, ลี้, ตำลึง (ไม่ใช่ zhang, li, tael)

### ✅ One-Pass Fixes (Common English-isms)
| ENG structure | ❌ Wrong | ✅ Right |
|--------------|----------|---------|
| "Only fourteen years old, his skin was..." | "อายุแค่สิบสี่ ผิว..." | "เขามีอายุสิบสี่ปี ผิว..." |
| "There was a carriage..." | มีรถม้าหนึ่งคัน... | รถม้าคันหนึ่ง... |
| "He was attacked by..." | เขาถูกโจมตีโดย... | เขาถูกโจมตี... (ตัด by) |
| "In order to..." | เพื่อที่จะ... | เพื่อง... |
| "Although... but..." | ถึงแม้ว่า...แต่... | แม้...ก็ตาม... |
